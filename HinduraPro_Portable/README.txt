Hindura Pro - File Converter
============================

check it [App] out in release page: https://github.com/honoreade/Hindura/releases

SETUP:
Double-click HinduraPro.exe to run!

FEATURES:
- Convert videos, audio, images
- Batch processing (multiple files at once)
- Resize and compress media
- Dark/Light theme

REQUIREMENTS:
- Windows 10/11
- ffmpeg.exe (in this folder)

Enjoy!
